# Visited Places

A simple web application that allows users to keep track of places they have visited. Each place stores basic information and displays details when selected.

## Project Description

This project is built to practice JavaScript constructor functions and prototype methods. Each place visited is created as an object with properties such as location, landmarks, best time to visit, and notes.

The application allows users to:

- View a list of places visited
- Click on a place name to see its details
- Understand how objects, constructors, and prototypes work in JavaScript
- Practice basic DOM manipulation
